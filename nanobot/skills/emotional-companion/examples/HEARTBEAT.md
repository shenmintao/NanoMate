# 情感伙伴定时任务

## 主动关怀检查

检查是否需要主动联系用户：

### 条件
- 用户超过24小时没有发消息 → 发送温和的问候
- 最近几次对话用户情绪都比较负面 → 主动提供支持
- 早上8-9点 → 发送早安问候
- 晚上9-10点 → 询问今天过得如何

### 执行步骤

1. **检查用户最后活跃时间**
   - 如果超过24小时，发送check-in消息
   - 如果在正常作息时间内，继续下一步

2. **检查情绪历史**
   - 回顾最近的对话
   - 如果发现连续的负面情绪（压力、焦虑、难过等），提供支持

3. **时间相关问候**
   - 当前时间8-9点：发送早安问候"早上好！希望今天一切顺利☀️"
   - 当前时间21-22点：发送晚安问候"今天过得怎么样？要不要一起回顾一下？"

4. **发送消息**
   使用 message 工具发送到用户的 WhatsApp（或其他渠道）：
   ```
   message(
     channel="whatsapp",
     chat_id="用户ID",
     content="问候或关怀消息"
   )
   ```

## 注意事项

- 不要过于频繁打扰用户（每天最多2-3次主动消息）
- 如果用户回复简短或表示忙碌，给予空间
- 保持自然，不要像机器人一样打卡
- 真诚关心，不是为了完成任务而问候
